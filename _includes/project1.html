Layouty:

  • page.html - základný layout pre stránku, ďalšie layouty ho rozširujú

  • model.html - layout pre jednotlivé plastové modely, ktoré staviam ho voľnom čase

  • post.html - layout pre príspevok blogu

  • project.html - layout pre zadanie

Kolekcie:

  • Životopis - pri tvorbe CV som využil 4 kolekcie:

    • r-education - dosiahnuté vzdelanie, kurzy ...

    • r-experience - pracovné skúsenosti

    • r-skills - technické zručnosti, programovacie jazyky ...

    • r-languages - cudzie jazyky

  • hobbies - hobby, záľuby, trávenie voľného času

  • projects - zadania z predmetu WP

Premenné:

V projekte využívam všetky druhy premenných.

  • Site - pri prístupe ku kolekciám

  • Page - pri generovaní stránok

  • Ostatné - pri vyberaní dát z dátových súborov (napr. CV)

Dátové súbory:

Dátové súbory využívam pri tvorbe blogu alebo pri tvorbe detailu plastových modelov(sekcia About a následne vybrať niektorý model). Príklad dátového súboru (_hobbies/hind.md):

Filtre / Tagy:

  • Upcase - string filter

  • Sort - pri zoraďovaní v rámci CV

  • Date to String - pri blogu

  • Smartify - pri blogu na dosiahnutie oblých úvodzoviek

  • Include - include headera a footra do layoutu

Pluginy:

  • Fetch remote file content - plugin na zobrazenie kódu z GitHubu

  • Jekyll Video Embed - plugin na zobrazenie videa (napr. z YouTube) na stránke
