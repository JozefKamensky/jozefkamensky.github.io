Na definovanie elementov som použil XML Schemu. Základné elementy a ich opis:

  • prezentacia - koreňový element ktorý obshauje elemetny meta a snimok
  • meta - element ktory obsahuje metadáta (autor) a nastavenia(číslovanie strán)
  • snimok - element reprezentujúci snímok prezentácie, skladá sa z nadpisu, podnadpisu a obsahu
  • obsah - element obsahujúci obsah daného snímku, číslo za ním, napr. obsah100, označuje šírku v percentách
  • obrazovyObsah - element reprezentujuci obrazovy obsah, obsahue atribut zdroj - URI zdroja
  • textovyObsah - element reprezentujúci textový obsah, obsahuje prvky text, zoznam alebo link
  • text - element so samotným textom, ktorý môže byť doplnený o tagy zvýrazňujúce text - tučné. kurzíva a podčiarknutie. Tiež je možné pomocou atribútov velkost a farbaHEX nastaviť veľkosť, resp. farbu
  • zoznam - element pre zoznamy (podľa atribútu číslované alebo nie), skladá sa z položiek alebo ďalších zoznamov
  • položka - element položka v sebe obsahuje textovyObsah
  • link - element pre zobrazenie odkazov, napr. na internetové stránky

Farba textu na zadáva v HEXa formáte a kontroluje sa regexom "^#[A-Fa-f0-9]{6}$", ktorý zabezpečuje že bude obsahovať len povolené znaky a bude správnej dĺžky.
